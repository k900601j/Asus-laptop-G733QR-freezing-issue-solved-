# Asus-Laptop-G733QR-freezing-issue-solved-
# 1. I can't install windows from my laptop because it keeps stopping. So I install Windows 11 from another computer and boot to the installed windows to download and run Ghelper.
# 2. Ghelper settings : Performance Mode-Turbo, gpu Mode-Optimization, Screen-Automatic, and Run at System Startup.
# 3. Now, after I finish setting up windows and ghelper, I take out the ssd and install it on the G733QR Laptop.
# 4. Turn on the power and press the F2 key to enter the bios : Animation Post Logo Audio disable, disable the svm, disable the ASUS Armoury Crate, disable the secure boot, Reboot saving changes by pressing F10
# 5. After booting to a G733QR laptop, first install the built-in graphics driver for the 5900HX CPU using AMD adrenaline (driver only), then download and install the G733QR graphics driver and other drivers from the AMD homepage, and then install all drivers using MYASUS for drivers that are not installed. Finally, disable Windows updates
